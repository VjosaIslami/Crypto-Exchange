Available API Endpoints:

  • GET /api/market-data - Market statistics
  • GET /api/symbols - Available trading symbols
  • GET /api/technical-analysis?symbol=BTCUSDT - Technical analysis
  • GET /api/lstm-prediction?symbol=BTCUSDT - LSTM price prediction
  • GET /api/sentiment-analysis?symbol=BTCUSDT - Sentiment analysis
  • POST /api/update - Trigger data update
  • GET /api/export/csv?symbols=BTCUSDT,ETHUSDT - Export CSV
  • GET /api/health - Health check for all services
